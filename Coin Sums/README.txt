#Coin Sum

##Overview

Returns the number of ways to create a certain value in pence using all possible denominations of coins.
Denominations available: 1p, 2p, 5p, 10p, 20p, 50p, 1 pound, 2 pound.
Asks the user for an input amount in pence as the certain value. Outputs all possible ways to create user's amount using all denominations of coins.

##Compiling

Once within the correct directory, enter:

g++ coin_sum.cpp -o coin_sum -Wall -Werror

##Running

Run with:

./coin_sum
