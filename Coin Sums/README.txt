Coin Sum

Description:	Returns the number of ways to create a certain value in pence using all possible denominations of coins.
				Denominations available: 1p, 2p, 5p, 10p, 20p, 50p, 1 pound, 2 pound
				To run the program, enter the commands below in command line once within the "Coin Sum" directory.

1) Compiling

Enter: g++ coin_sum.cpp -o coin_sum -Wall -Werror

2) Running

Enter: ./coin_sum
