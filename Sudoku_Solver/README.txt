Sudoku Solver

Description:	Solves any sudoku and outputs the result.

1) Compiling

Enter: g++ sudoku_solver.cpp -o sudoku_solver -Wall -Werror

2) Running

Enter: ./sudoku_solver

Fill in the preset values using indexing for rows and columns.
	Eg. The top left corner would have coordinates row: 0, col: 0.
	Eg. The top right corner would have coordinates row: 0, col: 8
	Eg. The bottom left corner would have coordinates row: 8, col: 0
	Eg. The bottom right corner would have coordinates row: 8, col: 8
